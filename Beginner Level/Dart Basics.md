## Syntax and Structure

Dart syntax is clean and straightforward. Below are some basic examples:

### Example 1: Basic Hello World

```dart
void main() {
  print('Hello, Dart!');
}
```

### Example 2: Variables and Basic Operations

```
void main() {
  var name = 'Dart';
  var age = 10;
  print('Name: $name, Age: $age');
}

```
